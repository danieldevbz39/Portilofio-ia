# Rodando o projeto localmente

Este repositório contém o front-end da aplicação. Para visualizar localmente sem dependências de IA:

## Requisitos

- Node.js (opcional, apenas para modo dev com Vite)

## Como visualizar (estática)

1. Abra `index.html` diretamente no navegador, ou sirva com um servidor estático:

```bash
cd "/home/danieldev/Downloads/larica-marcial com ia/larica"
python3 -m http.server 8000
# abra http://localhost:8000
```

## Como rodar em modo de desenvolvimento (opcional)

1. Instale dependências:
   `npm install`
2. Rode o servidor Vite:
   `npm run dev`

Nota: chaves de API para serviços de IA foram removidas deste repositório para evitar cobranças acidentais.
